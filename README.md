## What it covers

Five sections, in this order:

1. **Tips & Tricks** — community + official prompting and workflow tips
2. **Feature Spotlight** — one Claude feature per week, with linked guides
3. **Claude Releases** — model + product + API updates from Anthropic
4. **Community & Open Source** — plugins, MCP servers, threads worth reading
5. **AI / LLM Landscape** — OpenAI, Google, Meta, Mistral, DeepSeek, benchmarks, policy

## Cadence

Published every Friday, covering the previous seven days. Drafted by Claude, curated by a human.

## Local viewing

Open any `.html` file directly in a browser — they're self-contained static pages with light/dark theme toggle (auto-detects system preference on first load).
