Tips & Tricks
Marmelab · Apr 24
Use /rewind instead of continuing after a bad response. A wrong answer stays in context and pollutes everything that follows. /rewind rolls back to the last good state; /clear nukes the whole context. Both are cheaper than arguing with Claude about its mistake. All tips →
Plugin recommendation
Install an LSP plugin — it's the highest-impact single add-on. LSP feeds Claude automatic diagnostics after every file edit, so it sees and fixes type errors, lint warnings, and import issues before you even notice them. Pair it with Context7 for indexed library docs at exact versions — no more Claude hallucinating API signatures.
Prompting technique
Use @ to load files directly into context. Without it, Claude has to search for the file and read it in chunks — slower and uses more tokens. @src/auth/middleware.ts loads the entire file immediately. Combine with XML tags for multi-file prompts: wrap each file reference in a labeled section so Claude knows what each one is for.

Claude Code Hooks
Hooks are user-defined shell commands that fire at specific lifecycle points — before/after tool calls, on session start/end, and more. Think Git hooks for your AI coding workflow. Define the event, the matcher, and the action; Claude executes them deterministically every time, no prompting required. 25 lifecycle events as of the latest release, with async and HTTP hooks for advanced integrations.
