Tips & Tricks
Install the security plugin before your next session · Anthropic
The new security-guidance plugin flags 25 dangerous patterns as you type. Injection, unsafe deserialization, sketchy DOM calls. It runs automatically once installed, no separate tool, no extra command. The instant checks run without a model call, so they cost nothing. Security guidance →
Review the diff in a fresh subagent · Claude Code best practices
Before you call a task done, run /code-review so a subagent checks the diff in a clean context. It sees only the change and your criteria, not the reasoning that produced it. The longer Claude works unattended, the more that independent pass earns its place. Best practices →
Reach for subagents first, skills second · Tembo
Need Claude to read 30 files? Spawn a subagent instead of dumping them into the main thread. The subagent does the heavy reading in its own context and hands back a short summary. Move to skills when you want the same instructions inline. Note: subagents cannot spawn subagents. Subagents guide →
Let Memory learn your stack · Claude Help Center
Memory now ships on every account, free tier included. Claude distills durable facts from your chats roughly every 24 hours. Role, tools, recurring project context. Then loads them into each new conversation. Stop re-explaining your setup at the top of every thread. How memory works →

Code Review, when the second pair of eyes is also Claude
Claude Code's review tools read your diff in a fresh context and report what is wrong before a human ever looks. The new security-guidance plugin runs three layers. Instant pattern checks with no model call. A deeper scan after AI-generated edits. A logic-flaw pass at commit time. It catches 25 high-risk patterns, from injection to unsafe deserialization. Anthropic measured a 30 to 40 percent drop in security comments on PRs opened with it. Reach for review when work ran unattended, or before you ship anything you did not write line by line. The reviewer sees the change, not the story behind it.
